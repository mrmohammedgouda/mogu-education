## 🛠️ الأوامر السريعة

### **فحص الموقع:**
```bash
curl -I https://moguedu.ca
```

### **نسخة احتياطية:**
```bash
cd /home/user/webapp
bash backups/backup-database.sh local
```

### **تحديث الموقع:**
```bash
cd /home/user/webapp
git pull origin main
npm run build
npx wrangler pages deploy dist --project-name moguedu
```

### **فحص قاعدة البيانات:**
```bash
npx wrangler d1 execute moguedu-production --local \
  --command="SELECT COUNT(*) FROM certificates"
```

## 🚨 حالات الطوارئ

### **الموقع لا يعمل؟**
```bash
1. تحقق من: https://www.cloudflarestatus.com/
2. تحقق من DNS: nslookup moguedu.ca
3. راجع: https://dash.cloudflare.com
4. أعد النشر إذا لزم الأمر
```

### **فقدان البيانات؟**
```bash
cd /home/user/webapp
npx wrangler d1 execute moguedu-production --remote \
  --file=backups/moguedu-backup-latest.sql
```

### **نسيت كلمة مرور Admin؟**
```bash
# إعادة تعيين إلى admin123
npx wrangler d1 execute moguedu-production --remote \
  --command="UPDATE admin_users SET password = 'admin123' WHERE username = 'admin'"
```
